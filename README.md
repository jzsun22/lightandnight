# 光夜小工具
北极光工作室手机游戏《光与夜之恋》相关小工具集合，包含各类计算器与抽卡模拟器。

玩家为爱发电产物，非官方，且与任何其他玩家组织无关。

数据均来自官方公示，需手动同步更新，可能存在更新延迟、数据错误、跑路等风险，请以游戏中实际情况为准。
